## Set up env
Create env
```bash
python3 -m venv venv
```
Activate env
```bash
source venv/bin/activate
```
Deactivate env
```bash
deactivate
```

## Install requirements
```bash
pip install -r requirements.txt
```

## Update requirements
```bash
pip freeze > requirements.txt
```
